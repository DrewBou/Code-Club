## Control Structures ##

Control structures allow parts of a program to run in a non-sequential order.

### Decision-Making Control Structures ###

- These include if, if...else, and switch statements (Check IfElse folder)
  - if statement:

  ```
  if (this == true) {
    //run this code
  }
  ```

  - if...else statement:

  ```
  if (this == true) {
    //run this code
  }

  else {
    //run this code
  }
